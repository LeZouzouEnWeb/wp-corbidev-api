# docs — WP Corbidev API (generic plugin)

Ce dossier contient la documentation pour transformer le plugin en plugin générique multi‑API.

- Objectif: découvrir des dossiers `Pages_<slug>` et charger dynamiquement les pages/admin/endpoints.
- Emplacement: `Pages_<slug>/manifest.json` ou `Pages_<slug>/Pages.php`.
- Voir `Pages-conventions.md` pour la structure du manifest.
